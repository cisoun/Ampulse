Ampulse
=======

Guitare amplifier and cabinet simulator.

## Features

* Pregain
* 10 bands equalizer
* Distortion
* Reverb
* Master
* Cab simulator (not working yet)
* Microphone selector

## Depedencies

* CSCore (from NuGet)

## Installation

1) Compile the project with Visual Studio.
2) Move the doc/IR folder in the folder where the new executable is (bin).
